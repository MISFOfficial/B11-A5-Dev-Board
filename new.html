Question-1: What are the different ways to select an element in the DOM?

Ans : Hare is the primary methods to select element in the DOM

  • document.getElementById('element')
  • document.getElementByClassName('element')
  • document.querySelector('element')
  • document.querySelectorAll('element')

    Question-4: What is event bubbling in the DOM?

    Ans : Event Bubbling is a concept in the DOM (Document Object Model). It happens when an element receives an event, and that event bubbles up to its parent and ancestor elements in the DOM tree until it gets to the root element.
